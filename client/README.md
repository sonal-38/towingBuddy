# TowingBuddy Client

Frontend React application for the TowingBuddy vehicle towing management system.

## Features

- **Admin Panel**: Manage towing records, dashboard analytics
- **User Panel**: Vehicle tracking, fine payments, history
- **Modern UI**: Professional design with Tailwind CSS
- **Responsive**: Works on desktop and mobile devices

## Tech Stack

- React 18 + TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- Shadcn/ui components
- React Router for navigation
- React Query for state management

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
